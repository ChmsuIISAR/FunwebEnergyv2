# EnergyFlow Simulations

A mobile-first, canvas-based educational platform visualizing energy transfer and transformation without complex physics formulas.

## Tech Stack
- **Framework**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   ```

3. **Open in browser**
   Navigate to `http://localhost:5173`

## Project Structure
- `simulations/`: Individual physics simulation components.
- `components/`: Shared UI components (Controls, Canvas wrapper).
- `types.ts`: Shared type definitions.
